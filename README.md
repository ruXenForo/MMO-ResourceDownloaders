Русский язык плагина MMO Who Downloaded This Resource для XenForo 2

## Файлы
* [XenForo](https://xenforo.com/)
* [MMO-ResourceDownloaders](https://mmo-zone.info/dbtech-ecommerce/mmo-who-downloaded-this-resource.15/)
